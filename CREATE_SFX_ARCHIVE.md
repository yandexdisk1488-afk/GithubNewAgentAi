# 📦 Инструкция: Как создать SFX архив ChatGPT GUI

## Что такое SFX архив?
**SFX (Self-Extracting Archive)** - это самораспаковывающийся архив, который сразу запускается при открытии, не требуя внешних программ распаковки.

---

## ✅ Шаг 1: Подготовка файлов

Создай папку для архива:
```
📁 ChatGPT_GUI_SFX/
├── 📄 ChatGPT_GUI.py (основной файл)
├── 📄 requirements.txt
├── 📄 README.md
└── 📄 run.bat (для Windows)
```

---

## 📝 Шаг 2: Создай `requirements.txt`

```txt
PyQt6>=6.5.0
openai>=1.3.0
python-dotenv>=1.0.0
requests>=2.31.0
```

**Сохрани как:** `requirements.txt`

---

## 🪟 Шаг 3: Создай `run.bat` (для Windows)

```batch
@echo off
chcp 65001 >nul
echo ========================================
echo 🚀 ChatGPT GUI - Автозапуск
echo ========================================
echo.

REM Проверяем наличие Python
python --version >nul 2>&1
if errorlevel 1 (
    echo ❌ Python не установлен!
    echo Установи Python с https://python.org
    pause
    exit /b 1
)

echo ✅ Python найден

REM Устанавливаем зависимости
echo.
echo 📦 Установка зависимостей...
pip install -q -r requirements.txt

if errorlevel 1 (
    echo ❌ Ошибка установки зависимостей!
    pause
    exit /b 1
)

echo ✅ Зависимости установлены

REM Запускаем приложение
echo.
echo 🎬 Запуск приложения...
python ChatGPT_GUI.py

pause
```

**Сохрани как:** `run.bat`

---

## 🐧 Шаг 4: Создай `run.sh` (для Linux/Mac)

```bash
#!/bin/bash

echo "========================================"
echo "🚀 ChatGPT GUI - Автозапуск"
echo "========================================"
echo ""

# Проверяем наличие Python
if ! command -v python3 &> /dev/null; then
    echo "❌ Python не установлен!"
    echo "Установи: sudo apt-get install python3 python3-pip"
    exit 1
fi

echo "✅ Python найден"

# Устанавливаем зависимости
echo ""
echo "📦 Установка зависимостей..."
pip3 install -q -r requirements.txt

if [ $? -ne 0 ]; then
    echo "❌ Ошибка установки зависимостей!"
    exit 1
fi

echo "✅ Зависимости установлены"

# Запускаем приложение
echo ""
echo "🎬 Запуск приложения..."
python3 ChatGPT_GUI.py
```

**Сохрани как:** `run.sh` и выполни:
```bash
chmod +x run.sh
```

## 🎁 Шаг 6: Создание SFX архива

### Способ 1: WinRAR (рекомендуется)

1. **Скачай WinRAR** с https://www.rarlab.com/
2. Выбери все файлы в папке `ChatGPT_GUI_SFX`
3. Нажми **правая кнопка мыши** → **Добавить в архив**
4. В окне настроек:
   - ✅ Установи **"Создать SFX архив"**
   - ✅ **Вкладка "Дополнительно"** → **"SFX параметры"**
   - Установи:
     - **Путь распаковки:** `.` (текущая папка)
     - **Команда после распаковки:** `run.bat` (для Windows) или `./run.sh` (для Linux)
     - ✅ **"Скрыть всё"** (опционально)
5. Нажми **OK** и архив будет создан

### Способ 2: 7-Zip (бесплатно)

1. **Скачай 7-Zip** с https://www.7-zip.org/
2. Выбери все файлы в папке `ChatGPT_GUI_SFX`
3. **Правая кнопка** → **7-Zip** → **Добавить в архив**
4. Выбери тип архива: **7z**
5. Создай архив

> ⚠️ 7-Zip создает обычный архив, но можно конвертировать в EXE

### Способ 3: Python (автоматический)

Создай файл `create_sfx.py`:

```python
import zipfile
import os
from pathlib import Path

def create_sfx_archive():
    """Создать SFX архив"""
    folder = Path("ChatGPT_GUI_SFX")
    output = "ChatGPT_GUI_v1.0_SFX.zip"
    
    print(f"📦 Создание архива: {output}")
    
    with zipfile.ZipFile(output, 'w', zipfile.ZIP_DEFLATED) as zf:
        for file in folder.rglob("*"):
            if file.is_file():
                arcname = file.relative_to(folder)
                zf.write(file, arcname)
                print(f"  ✅ {arcname}")
    
    print(f"✅ Архив успешно создан: {output}")

if __name__ == "__main__":
    create_sfx_archive()
```

Запусти:
```bash
python create_sfx.py
```
