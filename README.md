# 🤖 GitHub New Agent AI

Полнофункциональный AI агент на основе OpenAI API (ChatGPT) написанный на Python.

## ✨ Возможности

- 💬 Интерактивный чат с AI
- 🔄 Сохранение истории разговора
- 🎯 Кастомизируемые системные промпты
- 🌡️ Регулировка температуры (детерминированность ответов)
- 📝 Просмотр истории разговора
- 🧹 Очистка истории
- 🛡️ Обработка ошибок

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/yandexdisk1488-afk/GithubNewAgentAi.git
cd GithubNewAgentAi
```

### 2. Создание виртуального окружения (рекомендуется)
```bash
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка API ключа

Создайте файл `.env` в корне проекта:
```bash
cp .env.example .env
```

Откройте `.env` и добавьте ваш OpenAI API ключ:
```
OPENAI_API_KEY=sk-...your-key-here...
```

**Где получить API ключ?**
1. Перейдите на https://platform.openai.com/api-keys
2. Создайте новый API ключ
3. Скопируйте его в файл `.env`

### 5. Запуск агента
```bash
python3 ai_agent.py
```

## 🔧 Продвинутое использование

### Использование в собственном коде

```python
from ai_agent import AIAgent

# Создание агента
agent = AIAgent(model="gpt-3.5-turbo", temperature=0.7)

# Кастомный системный промпт
agent.set_system_prompt("Ты программист на Python. Помогай пользователю писать код.")

# Отправка сообщения
response = agent.chat("Напиши функцию для сортировки списка")
print(response)

# Получение длины истории
print(f"Сообщений в истории: {agent.get_conversation_length()}")

# Очистка истории
agent.clear_history()
```

## ⚙️ Параметры

### Модели OpenAI
- `gpt-4` - Самая мощная модель (требует отдельного доступа)
- `gpt-4-turbo-preview` - Быстрая версия GPT-4
- `gpt-3.5-turbo` - Быстрая и дешевая модель (по умолчанию)

### Температура
- `0.0` - Детерминированные ответы, без креативности
- `0.7` - Сбалансированные ответы (по умолчанию)
- `1.0+` - Более креативные и непредсказуемые ответы

## 🐛 Решение проблем

### "OPENAI_API_KEY не установлен"
- Убедитесь, что создан файл `.env`
- Проверьте, что в нем правильно указан API ключ
- Перезагрузитесь в Python или перезапустите программу

### "Ошибка при запросе к OpenAI"
- Проверьте интернет соединение
- Убедитесь, что у вас есть средства на счете OpenAI
- Проверьте, что API ключ валидный

### Программа медленно отвечает
- Это нормально при первом запросе
- Проверьте ваше интернет соединение
- Более сложные запросы требуют больше времени
